# Taller1
Sistema de Gestión de Cursos Online

En muchos entornos educativos, la información de estudiantes, instructores y cursos se maneja de forma desorganizada o manual, lo que dificulta el registro, la consulta y la actualización de los datos.

Esta situación puede generar pérdida de información, duplicidad de registros y problemas al momento de gestionar las inscripciones de los estudiantes en los cursos.

Por esta razón, se propone desarrollar un sistema básico que permita administrar de manera sencilla los cursos, los usuarios y sus inscripciones, facilitando las operaciones de crear, consultar, actualizar y eliminar registros.

El sistema será implementado como una aplicación de consola en C#, utilizando archivos CSV para almacenar la información
